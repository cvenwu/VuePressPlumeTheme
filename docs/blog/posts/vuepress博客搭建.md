## 环境介绍
1. Node.js **v18.20.0+**
2. **pnpm 8+** 或 **Yarn 2+**

## 安装node

### 普通安装
点击[node官网](https://nodejs.org/zh-cn/download/package-manager)下载安装v18.20.4即可

## 安装pnpm
执行命令`npm install -g pnpm@8.15.9`安装8+版本的pnpm即可
